## 2. 기술 스택 상세

### 백엔드
| 항목 | 내용 |
|------|------|
| Framework | Spring Boot |
| ORM | Spring Data JPA (Hibernate) |
| Security | Spring Security + JWT (jjwt) |
| DB | MySQL |
| Build | Gradle |
| API 문서 | Springdoc OpenAPI (Swagger UI) |

### 프론트엔드
| 항목 | 내용 |
|------|------|
| Framework | React + Vite |
| Routing | React Router v6 |
| HTTP | Axios |
| 상태 관리 | Zustand |
| 스타일 | Tailwind CSS |

## 3. 개발 단계 로드맵

### Phase 1 — 기반 설계 (1주)
- [ ] DB 스키마 설계 및 MySQL 세팅
- [ ] Spring Boot 프로젝트 초기화 (패키지 구조, 공통 응답, 예외 처리)
- [ ] React + Vite 프로젝트 초기화 (라우터, Axios 인터셉터)
- [ ] JWT 인증 구조 설계

### Phase 2 — 인증/사용자 (1주)
- [ ] 회원가입, 로그인, 로그아웃, Refresh Token 구현
- [ ] 사용자 프로필 조회/수정
- [ ] Spring Security 역할 기반 권한 처리
- [ ] 로그인/회원가입 페이지 UI

### Phase 3 — 강의 CRUD (2주)
- [ ] 강의 등록/수정/삭제 (강사)
- [ ] 강의 목록/상세 조회 API (카테고리, 키워드, 정렬, 페이징)
- [ ] 섹션 CRUD
- [ ] 레슨 CRUD
- [ ] 강의 목록/상세 페이지 UI
- [ ] 강사 대시보드 UI

### Phase 4 — 수강/진도 (1주)
- [ ] 수강 신청 로직 (ENROLLMENT 생성)
- [ ] 진도율 관리 (레슨 완료 처리)
- [ ] 내 수강 목록 페이지

### Phase 5 — 리뷰 (0.5주)
- [ ] 리뷰 CRUD
- [ ] 평균 별점 자동 갱신
- [ ] 리뷰 UI

### Phase 6 — 결제 내역 (0.5주)
- [ ] 결제 내역 조회 API (수강생)
- [ ] 결제 내역 조회 API (관리자 전체)
- [ ] 결제 취소(환불) API (관리자)

> 실제 PG사 연동(토스페이먼츠 준비/확정/검증)은 P3 범위.
