# 강의 플랫폼 서비스 기획서

> 프로젝트명: **EditHub**
> 작성일: 2026-03-12
> 버전: v1.0 (P2 범위)

---

## 1. 프로젝트 개요

### 1.1 서비스 소개
EditHub는 영상 편집 강사가 직접 강의를 등록하고, 수강생이 수강할 수 있는 **영상 편집 전문 온라인 강의 플랫폼**이다.
강의 영상, 강의 자료, 수강 진도 관리, 리뷰 시스템을 제공한다.

### 1.2 서비스 목표
- 강사가 손쉽게 강의를 개설하고 관리할 수 있는 환경 제공
- 수강생이 체계적으로 강의를 수강하고 진도를 관리할 수 있도록 지원
- 관리자가 서비스 전반을 모니터링하고 운영할 수 있는 대시보드 제공

### 1.3 타깃 사용자
| 사용자 유형 | 설명 |
|------------|------|
| 강사 (INSTRUCTOR) | 강의를 제작하고 등록하는 사용자 |
| 수강생 (STUDENT) | 강의를 구매하고 수강하는 사용자 |
| 관리자 (ADMIN) | 서비스 전체를 관리하는 운영자 |

## 2. 핵심 기능

### 2.1 사용자 기능 (P2)
- 회원가입 / 로그인 (JWT 기반 인증)
- 프로필 조회 및 수정
- 비밀번호 변경

### 2.2 강의 기능 (P1 + P2)
- 강의 목록 조회 (카테고리 필터, 검색, 정렬)
- 강의 상세 조회 (소개, 커리큘럼, 강사 정보, 리뷰)
- 강의 수강 신청 (무료 강의 즉시 수강)

### 2.3 강사 기능 (P2)
- 강의 등록 / 수정 / 삭제
- 강의 내 섹션(챕터) 및 강의 영상(레슨) 관리
- 수강생 현황 조회

### 2.4 수강생 기능 (P2)
- 내 강의 목록 조회
- 강의 진도율 관리
- 강의 리뷰 작성 / 수정 / 삭제

### 2.5 관리자 기능 (P2)
- 전체 회원 관리 (조회, 권한 변경, 정지)
- 전체 강의 관리 (승인, 비활성화)
- 결제 내역 전체 조회

---

## 3. 서비스 흐름

### 3.1 강의 등록 흐름 (강사)
```
강사 로그인 -> 강의 등록 (제목, 소개, 카테고리, 가격, 썸네일)
-> 섹션 추가 -> 레슨(영상) 추가 -> 강의 공개 요청
-> 관리자 승인 -> 수강생에게 노출
```

### 3.2 강의 수강 흐름 (수강생)
```
강의 목록 탐색 -> 강의 상세 조회 -> 수강 신청
-> 수강 시작 -> 진도율 업데이트 -> 수강 완료
-> 리뷰 작성 (선택)
```

## 5. 기술 스택

| 분류 | 기술 |
|------|------|
| 프론트엔드 | React 18, Vite, React Router v6, Axios, Tailwind CSS |
| 백엔드 | Spring Boot 3.x, Spring Security, JPA (Hibernate) |
| 데이터베이스 | MySQL 8.x |
| 인증 | JWT (Access Token + Refresh Token) |

---

## 6. 기술적 제약 사항
- JWT 만료 시 Refresh Token으로 자동 재발급
- 관리자 승인 없이 강의가 공개되지 않도록 처리
- 모든 API는 `/api` 접두사 통일
