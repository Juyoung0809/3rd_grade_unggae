# 09. 강사 수익 대시보드 — 설계서

> 기존 기획서·요구사항명세서에 없던 화면을 신규 설계함.
> 이 문서를 기준으로 백·프론트를 동일하게 구현한다.

---

## 1. 화면 목적

강사(INSTRUCTOR)가 자신의 강의 판매 현황을 한눈에 파악할 수 있도록 한다.

- 언제, 얼마의 수익이 발생했는지 (월별 집계)
- 누가 어느 강의를 얼마에 구매했는지 (무료·유료 모두 표시)
- 취소·환불된 건도 이력으로 조회 가능

## 3. UI 구성

### 3-1. 페이지 레이아웃

```

├──────────┼────────────────┼─────────────────────────────┤
│ 2026-05  │     30,000     │      5                      │

└──────────┴────────────────┴─────────────────────────────┘
```

### 3-2. 요약 카드 3개

| 카드 | 값 계산 방법 |
|------|-------------|
| **총 수익** | 내 강의 ACTIVE 수강 건 `paidPrice` 합계 |
| **이번 달 수익** | 이번 달 ACTIVE 수강 건 `paidPrice` 합계 |
| **총 판매 건수** | 내 강의 전체 수강 건 수 (ACTIVE + CANCELLED 모두) |

- 취소(환불)된 건은 수익에 포함하지 않음
- 무료 강의(`paidPrice = 0`)는 수익 0원으로 집계되나 판매 건수에는 포함

### 3-3. 월별 수익 테이블

- 최근 12개월을 내림차순으로 표시
- ACTIVE 수강 건만 집계 (취소된 건 제외)
- 해당 월에 판매가 없으면 행 미표시

### 3-4. 판매 내역 테이블

- 기본: 내 전체 강의의 수강 내역, 등록일 내림차순
- 강의명 드롭다운으로 특정 강의만 필터 가능
- 상태 표시: `ACTIVE` → "수강 중" (초록), `CANCELLED` → "취소됨" (회색)
- 금액이 0이면 "무료"로 표시
- 수강생 이름은 `nickname` 사용

---

## 4. 데이터 소스

- **수익 데이터**: `Enrollment` 엔티티
  - `Enrollment.course.instructor = 현재 로그인 강사`로 필터
  - `paidPrice`: 실제 결제 금액 (무료면 0)
  - `status`: ACTIVE / CANCELLED
  - `enrolledAt`: 등록 시각
- **수강생 정보**: `Enrollment.user.nickname`
- **강의 정보**: `Enrollment.course.title`, `Enrollment.course.id`

Payment 엔티티는 Phase 6 비활성화 상태이므로 Enrollment를 직접 사용한다.

## 5. API 명세

### 5-1. 요약 + 월별 통계

```
GET /api/instructor/revenue/summary
Authorization: Bearer {accessToken}
```

**Response 200**
```json
{
  "totalRevenue": 150000,
  "thisMonthRevenue": 30000,
  "totalSalesCount": 25,
  "monthlyStats": [
    { "year": 2026, "month": 5, "revenue": 30000, "salesCount": 5 },
    { "year": 2026, "month": 4, "revenue": 60000, "salesCount": 10 }
  ]
}
```

### 5-2. 판매 내역 목록

```
GET /api/instructor/revenue/sales?courseId={courseId}
Authorization: Bearer {accessToken}
```

- `courseId` 파라미터 없으면 전체 강의 내역 반환
- 등록일 내림차순 정렬

**Response 200**
```json
[
  {
    "enrollmentId": 1,
    "studentNickname": "홍길동",
    "courseId": 10,
    "courseTitle": "유튜브 영상 편집 기초",
    "paidPrice": 29000,
    "enrolledAt": "2026-05-01T10:30:00",
    "status": "ACTIVE"
  }
]
```

## 8. 예외 처리

| 상황 | 처리 방법 |
|------|----------|
| 강의가 없는 강사 | 빈 데이터 반환 (수익 0, 목록 비어 있음) |
| `courseId` 필터가 타 강사 강의 ID | 내 강의가 아니므로 결과 빈 배열 반환 (403 아닌 빈 배열로 처리) |
| 미인증 요청 | 401 |
| STUDENT/ADMIN 요청 | 403 |
